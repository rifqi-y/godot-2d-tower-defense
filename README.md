# Defense of the Village — Godot 4 (2D Tower Defense)

A simple, fast-paced 2D tower defense game made with Godot 4. Place towers, survive enemy waves, and protect the village crystal.

## Gameplay
- Build towers along the enemy path.
- Each wave gets tougher — plan your placements and upgrades.
- Defend the village crystal: if its health reaches 0, it’s game over.

## How to Run
1. Install [Godot 4](https://godotengine.org/).
2. Clone this repo and open `project.godot` in Godot.
3. Press `F5` to run. You can also open any map scene (e.g., `map_1.tscn`) and run it directly.

## Controls
- Left Click: place/select towers
- Right Click / Esc: cancel placement or open back

## Project Structure
- `Scenes/` — main scenes, maps, and UI
- `Resources/` — data/resources used by scenes and scripts
- `Singletons/` — autoloads and global scripts
- `Assets/` — art, audio, and game logo (place your logo at `Assets/logo.png`)

## Built With
- Godot 4 + GDScript
